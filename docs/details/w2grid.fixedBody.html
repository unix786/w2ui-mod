Indicates if the body of the grid is of fixed height.
Boolean, default = true
If the body of the grid is of fixed height, the height will be taken from the box the grid is in. Setting up the height of the box will define the height of the grid. However, if you set .fixedBody to false, the grid will have dynamic height depending on the number of records in it.
