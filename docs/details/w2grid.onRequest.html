Called on any server request.
onRequest = function(event)
You can add event listener during the object creation: or after the object has been created: One of the important features of this event is that you can change the data that will be sent to the server. When the event is emitted, the postData property inside event object will have all the variables to be sent to the server. If you modify postData, the grid will send modified version to the server.
You can also change url property of event and the request will be sent to the new url.
You can also change httpHeaders property of event to change the AJAX headers of the request.
The event handler is called before the default action of the event is triggered. You can cancel the default action by calling event.preventDefault(). To perform an action after the event is fully processed, define event.onComplete function.
See events page in utilities for more details.
